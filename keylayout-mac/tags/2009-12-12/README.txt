WHATSTHIS

This is version 2009-12-12 of the Tengwar Keyboard Layout for Mac OS X, an additional tool from the Free Tengwar Fonts Project. For more information, see:

http://freetengwar.sourceforge.net/keylayouts-mac.html


INSTALLATION

Move any pair of a KEYLAYOUT file and a corresponding ICNS file to either one of the following folders:

'/Users/(your User name)/Library/Keyboard Layouts/'
'/Library/Keyboard Layouts/'

Then restart or log out to complete the installation. Then select the installed keyboard layout from the Input Sources tab of the International Preference Pane in the System Preferences application.
