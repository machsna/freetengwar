Introduction

Recent versions of all major web browsers support the embedding of fonts through the CSS3 @font-face selector. Windows Internet Explorer does so since version 4, Mozilla Firefox since version 3.5, Safari since version 3.1, and Opera since version 10.

Font embedding is especially useful for tengwar fonts, since they are not standardized. There are three options for using a non-standardized font on your website: You can convert all instances of the font into pictures, you can rely on your website’s visitors to install the font on their computers, or you can embed the font. Each option has its drawbacks. Converting all instances of the font into pictures makes the website heavy on pictures and inflexible, relying on your website’s visitors to install the font requires heavy interactions from the visitors, and embedding the font will not work on older web browsers.
